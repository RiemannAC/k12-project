# AlphaCamp DemoDay K12-Project

- 幫助 K12 教師訂定教學計劃，迅速檢視學期計劃、安排課程、待辨事項
- 檢視當日所需教材、教案，迅速歸檔及調用

## Install project
```
 $ git clone https://github.com/RiemannAC/k12-project.git
 $ cd k12-project
 $ bundle install
 $ rails db:migrate
```

## Generate seed data and fake user
```
 $ rails db:seed
 $ rails dev:fake_user
```
